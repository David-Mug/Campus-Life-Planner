Campus Life Planner - Regex Tests

Title Validation

Pattern: /^\S(?:.*\S)?$/

Description: No leading/trailing spaces

Duration Validation

Pattern: /^(0|[1-9]\d*)(\.[0-9]{1,2})?$/

Description: Numbers with optional decimal (up to 2 places)

Date Validation

Pattern: /^\d{4}-(0[1-9]|1[0-2])-(0[1-9]|[12]\d|3[01])$/

Description: YYYY-MM-DD format

Tag Validation

Pattern: /^[A-Za-z]+(?:[ -][A-Za-z]+)*$/

Description: Letters, spaces, hyphens

Advanced Regex: Duplicate Words

Pattern: /\b(\w+)\s+\1\b/

Description: Finds duplicate words
